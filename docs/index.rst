Chipyard 문서에 오신 것을 환영합니다 (버전 "|version|")!
==========================================================

.. image:: ./_static/images/chipyard-logo.svg

Chipyard는 애자일 팀을 사용하여 전체 시스템 하드웨어를 설계하고 평가하기 위한 프레임워크입니다.
이 프레임워크는 시스템 온 칩 개발을 위해 오픈 소스와 상용 도구 간의 통합을 제공하도록 설계된 도구와 라이브러리 모음으로 구성되어 있습니다.
이 작업은 NSF CCRI ENS Chipyard Award #201662의 지원을 받았습니다.

.. IMPORTANT:: **Chipyard를 처음 사용하시나요?** 설정 지침은 :ref:`Chipyard-Basics/Initial-Repo-Setup:Initial Repository Setup` 페이지로 이동하세요.
